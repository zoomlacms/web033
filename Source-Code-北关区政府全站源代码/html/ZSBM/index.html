我区召开“蓝天工程”攻坚动员会议

发布时间：2015-05-21 浏览次数：

5月9日上午11时，市“蓝天工程”攻坚动员大会结束后，我区及时在政府东二楼会议室召开了“蓝天工程”攻坚会议。区委书记程利军、常务副区长宁红亮、副区长华德强和有关单位一把手参加了会议。程书记要求：一是对我区存在的问题，要严格按照市“蓝天工程”督导组要求进行整改到位。二是各职能部门要引起高度重视，在我市“摘牌”期间，要采取超常规措施，明确分包和驻厂监督员。三是各重点企业要积极整改，大力配合迎检工作。四是重点发挥督查作用，对整改不到位的单位，区督导组要严格进行责任追究。

当天下午，常务副区长宁红亮和华德强副区长在建环局、彰北、园区主要领导陪同下冒雨对外环料场料堆进行了检查，对存在的问题提出了高标准整改要求。（住建环保局）
